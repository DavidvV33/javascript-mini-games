# 🎮 Mini Games

A collection of small browser-based games built with vanilla JavaScript, HTML and CSS.

## 🕹️ Games

### 🔐 Crack the Safe
Guess the secret number between 1 and 50. You start with 50 points — each wrong guess costs you one point. Beat your highscore!

### 🐷 Pig Game
A two-player dice game. On each turn, roll the dice as many times as you like to accumulate points — but if you roll a 1, you lose all points for that turn! First to 20 wins.
